# Games

Mobile-friendly HTML games — no dependencies, no backend, no accounts.

## Games

- 🃏 **Blackjack Trainer** — 6-deck shoe with basic strategy coach
- ♠️ **Texas Hold'em Trainer** — Casino table game vs the house

## Play

Open the site in Safari on iPhone, tap a game, then Share → Add to Home Screen for app-like access.

## Add a New Game

1. Create a new folder: `my-game/index.html`
2. Add a card to `index.html` (the launcher)
3. Push to GitHub — Cloudflare Pages auto-deploys
